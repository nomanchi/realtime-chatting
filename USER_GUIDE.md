## 시작하기

### 접속 방법

1. 웹 브라우저에서 애플리케이션 주소로 접속합니다
   - 예: `http://localhost:4001` 또는 `https://your-domain.com`

2. 처음 접속하면 랜딩 페이지가 표시됩니다
   - **로그인**: 기존 계정으로 로그인
   - **회원가입**: 새 계정 만들기
   - **익명으로 계속하기**: 로그인 없이 채팅 시작

## 회원가입 및 로그인

### 회원가입

1. 메인 페이지에서 **"회원가입"** 버튼 클릭
2. 다음 정보를 입력합니다:
   - **사용자명**: 다른 사용자에게 표시될 이름 (2-30자)
   - **이메일**: 로그인에 사용할 이메일 주소
   - **비밀번호**: 최소 6자 이상
   - **비밀번호 확인**: 비밀번호 재입력
3. **"회원가입"** 버튼 클릭
4. 회원가입이 완료되면 자동으로 로그인되어 채팅 화면으로 이동합니다

> 💡 **팁**: 이메일과 사용자명은 중복될 수 없습니다. 이미 사용 중인 경우 다른 값을 사용하세요.

### 로그인

1. 메인 페이지에서 **"로그인"** 버튼 클릭
2. 이메일과 비밀번호 입력
3. **"로그인"** 버튼 클릭
4. 로그인 성공 시 채팅 화면으로 이동합니다

### 로그아웃

- 채팅 화면 우측 상단의 **"로그아웃"** 버튼 클릭
- 또는 설정 페이지에서 로그아웃 가능

### 익명 채팅

로그인 없이도 채팅을 사용할 수 있습니다:

1. 메인 페이지에서 **"익명으로 계속하기"** 클릭
2. 이름 입력 (임시 사용자명)
3. **"채팅 참여하기"** 클릭

> ⚠️ **주의**: 익명 채팅의 경우:
> - 메시지가 데이터베이스에 저장되지 않습니다
> - 서버를 재시작하면 메시지가 사라집니다
> - 친구 기능을 사용할 수 없습니다

## 채팅 사용하기

### 기본 채팅 (Browser/WebView)

#### 메시지 보내기

1. 화면 하단의 입력창에 메시지 입력
2. **Enter** 키를 누르거나 전송 버튼 클릭
3. 메시지가 즉시 전송되어 모든 사용자에게 표시됩니다

#### 이미지 전송

1. 입력창 옆의 **첨부 버튼** 클릭
2. 이미지 선택
3. 이미지가 자동으로 전송됩니다

#### 이모지 사용

1. 입력창 옆의 **이모지 버튼** 클릭
2. 원하는 이모지 선택
3. 메시지에 이모지가 추가됩니다

### 채팅방 기능

#### 채팅방 목록 보기

1. 하단 네비게이션에서 **"채팅"** 탭 클릭
2. 참여 중인 모든 채팅방 목록이 표시됩니다
3. 각 채팅방에는 다음 정보가 표시됩니다:
   - 채팅방 이름 (또는 상대방 이름)
   - 마지막 메시지 미리보기
   - 읽지 않은 메시지 개수 (빨간 배지)
   - 마지막 메시지 시간

#### 채팅방 열기

- 채팅방 목록에서 원하는 채팅방을 **탭**하여 열기

#### 새 채팅방 만들기

1. 채팅방 목록 화면에서 **"+"** 버튼 클릭
2. 채팅방 유형 선택:
   - **1:1 채팅**: 친구와 1:1 대화
   - **그룹 채팅**: 여러 사람과 함께 대화
3. 참여할 사용자 선택
4. 그룹 채팅인 경우 채팅방 이름 입력
5. **"생성"** 버튼 클릭

#### 채팅방 이름 변경

1. 채팅방 목록에서 채팅방을 **길게 누르기** (Long Press)
2. 이름 입력창이 나타나면 새 이름 입력
3. **"저장"** 버튼 클릭

> 💡 **팁**: 1:1 채팅의 경우 상대방 이름이 기본으로 표시되지만, 원하는 이름으로 변경할 수 있습니다.

## 친구 기능

### 친구 목록 보기

1. 하단 네비게이션에서 **"친구"** 탭 클릭
2. 현재 친구 목록이 표시됩니다

### 친구 추가하기

#### 방법 1: 사용자 검색

1. 친구 페이지에서 **검색 아이콘** 클릭
2. 사용자명 또는 이메일로 검색
3. 검색 결과에서 원하는 사용자 선택
4. **"친구 요청"** 버튼 클릭

#### 방법 2: 프로필에서

1. 사용자 프로필 페이지로 이동
2. **"친구 요청"** 버튼 클릭

### 친구 요청 관리

1. 설정 페이지 → **"친구 요청"** 메뉴 클릭
2. 받은 친구 요청 목록 확인
3. 각 요청에 대해:
   - **"수락"**: 친구로 추가
   - **"거절"**: 요청 거절

### 친구와 채팅하기

1. 친구 목록에서 친구 선택
2. 자동으로 1:1 채팅방이 생성되거나 기존 채팅방이 열립니다

## 채팅방 관리

### 읽지 않은 메시지

- 채팅방 목록에서 읽지 않은 메시지가 있으면 빨간 배지로 개수가 표시됩니다
- 채팅방을 열면 자동으로 읽음 처리됩니다

### 메시지 히스토리

- 채팅방을 열면 최근 100개의 메시지가 자동으로 로드됩니다
- 스크롤을 위로 올리면 더 많은 메시지를 불러올 수 있습니다

### 채팅방 나가기

> ⚠️ 현재 버전에서는 채팅방 나가기 기능이 지원되지 않습니다. 향후 업데이트 예정입니다.

## 프로필 설정

### 내 프로필 보기

1. 하단 네비게이션에서 **"프로필"** 탭 클릭
2. 내 프로필 정보 확인:
   - 사용자명
   - 이메일
   - 아바타 (프로필 사진)

### 프로필 수정

1. 프로필 페이지에서 **"편집"** 버튼 클릭
2. 수정 가능한 정보:
   - 사용자명
   - 아바타 (프로필 사진)
3. 변경 사항 저장

### 다른 사용자 프로필 보기

1. 채팅방에서 사용자 이름 클릭
2. 또는 친구 목록에서 친구 선택
3. 사용자 프로필 페이지에서:
   - 사용자 정보 확인
   - 친구 요청 보내기 (친구가 아닌 경우)
   - 1:1 채팅 시작

## 자주 묻는 질문

### Q1. 메시지가 전송되지 않아요

**해결 방법:**
1. 인터넷 연결 확인
2. 페이지 새로고침 (F5)
3. 서버가 실행 중인지 확인

### Q2. 친구 요청이 보이지 않아요

**확인 사항:**
1. 설정 → 친구 요청 메뉴 확인
2. 이미 수락/거절한 요청은 표시되지 않습니다
3. 상대방이 요청을 취소했을 수 있습니다

### Q3. 채팅방 이름을 변경할 수 없어요

**해결 방법:**
1. 채팅방 목록에서 채팅방을 **길게 누르기** (Long Press)
2. 모바일에서는 1초 이상 길게 누르세요
3. PC에서는 우클릭 메뉴를 사용하세요

### Q4. 이전 메시지를 볼 수 없어요

**설명:**
- 익명 채팅의 경우 메시지가 저장되지 않아 이전 메시지를 볼 수 없습니다
- 로그인하여 사용하면 모든 메시지가 저장됩니다

### Q5. 여러 기기에서 동시에 사용할 수 있나요?

**답변:**
- 네, 가능합니다. 같은 계정으로 여러 기기에서 로그인할 수 있습니다
- 모든 기기에서 실시간으로 메시지를 받을 수 있습니다

### Q6. 그룹 채팅에 몇 명까지 참여할 수 있나요?

**답변:**
- 현재 제한이 없습니다. 원하는 만큼 사용자를 추가할 수 있습니다
- 단, 너무 많은 사용자가 참여하면 성능에 영향을 줄 수 있습니다

## 문제 해결

### 연결 오류

**증상**: "연결 끊김" 또는 "서버에 연결할 수 없습니다"

**해결 방법:**
1. 인터넷 연결 확인
2. 서버 주소가 올바른지 확인
3. 방화벽 설정 확인
4. 페이지 새로고침

### 로그인 오류

**증상**: "이메일 또는 비밀번호가 올바르지 않습니다"

**해결 방법:**
1. 이메일과 비밀번호가 정확한지 확인
2. 대소문자 구분 확인
3. 비밀번호 재설정 (향후 기능)
4. 회원가입을 다시 시도

### 메시지가 사라져요

**증상**: 이전에 보낸 메시지가 보이지 않음

**원인 및 해결:**
- **익명 채팅 사용 중**: 메시지가 저장되지 않습니다. 로그인하여 사용하세요
- **서버 재시작**: 익명 채팅 메시지는 서버 재시작 시 사라집니다
- **로그인 사용자**: 로그인한 경우 메시지는 저장되며, 채팅방을 다시 열면 표시됩니다

### 성능 문제

**증상**: 앱이 느리거나 멈춤

**해결 방법:**
1. 브라우저 캐시 삭제
2. 다른 탭/프로그램 종료
3. 브라우저 재시작
4. 인터넷 연결 속도 확인
